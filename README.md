# Основы программной инженерии

## Задание

1. Склонируйте репозиторий.
2. Настройте локальную ветку _develop_ на отслеживание удалённой ветки _origin/develop_.
3. В ветке develop измените файл _hello_everybody.txt_, добавив своё имя в конец файла.
4. Локально сохраните изменения.
5. Отправьте изменения на удалённый репозиторий.
